# 内容摘要

使用你的语言理解能力来摘要长内容。

## 使用方法

### 摘要文件
1. 使用 `read_file` 读取文件
2. 在回复中摘要内容

### 摘要网页
1. 使用 `web_fetch` 获取网页
2. 摘要提取的内容

### 摘要文本
直接摘要用户提供的文本。

## 提示

- 保持摘要简洁但信息丰富
- 突出关键点和结论
- 使用项目符号提高可读性
- 根据用户请求调整长度
